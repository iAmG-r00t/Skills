## The Truth Tables

### NOT

| NOT | True? |
| --- | ----- |
| not False | True |
| not True | False |

### OR

| OR | True? |
| -- | ----- |
| True or False | True |
| True or True | True |
| False or True | True |
| False or False | False |

### AND

| AND | True? |
| --- | ----- |
| True and False | False |
| True and True | True |
| False and True | False |
| False and False | False |

### NOT OR

| NOT OR | True? |
| ------ | ----- |
| not (True or False) | False |
| not (True or True) | False |
| not (False or True) | False |
| not (False or False) | True |

### NOT AND

| NOT AND | True? |
| ------- | ----- |
| not (True and False) | True |
| not (True and True) | False |
| not (False and True) | True |
| not (False and False) | True |

### NOT EQUAL

| != | True? |
| -- | ----- |
| 1 != 0 | True |
| 1 != 1 | False |
| 0 != 1 | True |
| 0 != 0 | False |

### EQUAL

| == | True? |
| -- | ----- |
| 1 == 0 | False |
| 1 == 1 | True |
| 0 == 1 | False |
| 0 == 0 | True |
